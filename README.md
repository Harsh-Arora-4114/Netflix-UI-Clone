# Netflix UI Clone

## Overview

This project is a responsive **Netflix UI Clone** developed using **HTML, CSS, and JavaScript**. It recreates the look and feel of Netflix's homepage, featuring a dynamic movie banner, categorized movie sections, responsive navigation, and embedded movie trailers using external APIs.

The project focuses on frontend web development, responsive design, API integration, and interactive user experience.

## Objectives

- Build a responsive Netflix-inspired user interface
- Practice modern HTML, CSS, and JavaScript
- Integrate external APIs for dynamic movie data
- Display trending and categorized movies
- Embed movie trailers using YouTube
- Improve frontend development skills

## How to Run

### 1. Clone Repository

```bash
git clone https://github.com/yourusername/netflix-ui-clone.git

cd netflix-ui-clone
```

### 2. Configure API Keys

Create a configuration file and add your API keys:

- TMDB API Key
- YouTube Data API Key

### 3. Run Project

Simply open:

```
index.html
```

Or use **Live Server** in VS Code.
